üìöWhat are Loops?

Loops are programming structures that repeat a block of code multiple times until a specific condition is met. They help us avoid writing the same code repeatedly.

Loops ek programming structure hai jo code ko baar-baar repeat karta hai jab tak ki koi specific condition meet na ho jaye. Ye humein same code ko baar-baar likhne se bachata hai.

Analogy: Imagine you're brushing your teeth. You move the brush up and down repeatedly until your teeth are clean. That's exactly what a loop does - it repeats an action until the job is done!
Key Definition: A loop is a control structure that executes a block of code repeatedly based on a given boolean condition.

üîÅ1. For Loop

The for loop is used when you know exactly how many times you want to repeat something. It has three parts: initialization, condition, and increment/decrement.

For loop tab use hota hai jab aapko pata ho ki kitni baar repeat karna hai. Isme teen parts hote hain: initialization (shuruat), condition (shart), aur increment/decrement (badhana/ghatana).

Analogy: Like climbing stairs - you start at step 1, check if you've reached the top (condition), and move to the next step (increment) until you reach your destination!

Syntax:

for (initialization; condition; increment/decrement) {
    // code to be executed
}

Example: Print numbers 1 to 5

for (let i = 1; i <= 5; i++) {
    console.log("Number: " + i);
}

Example: Sum of first 10 numbers

let sum = 0;
for (let i = 1; i <= 10; i++) {
    sum += i;
}
console.log("Sum of 1 to 10: " + sum);

üîÑ2. While Loop

The while loop repeats code as long as a condition is true. Use it when you don't know exactly how many times to loop.

While loop tab tak code repeat karta hai jab tak condition true hai. Iska use tab karo jab aapko exact count nahi pata.

Analogy: Like filling a bucket with water - you keep pouring (loop) while the bucket is not full (condition). You don't count how many times you pour, you just check if it's full!

Syntax:

while (condition) {
    // code to be executed
    // don't forget to update the condition!
}

Example: Count from 1 to 5

let count = 1;
while (count <= 5) {
    console.log("Count: " + count);
    count++;
}
Important: Always update the loop variable inside a while loop, otherwise you'll create an infinite loop (loop that never ends)!

üîÉ3. Do...While Loop

The do...while loop is similar to while loop, but it executes the code at least once before checking the condition.

Do...while loop while loop jaisa hai, lekin ye code ko kam se kam ek baar zaroor execute karta hai, phir condition check karta hai.

Analogy: Like trying a new dish - you taste it first (do), then decide if you want more (while condition). You always take at least one bite!

Syntax:

do {
    // code to be executed
} while (condition);

Example: Execute at least once

let num = 10;
do {
    console.log("Number: " + num);
    num++;
} while (num < 5);
console.log("Loop ended. Final num: " + num);
Key Difference: while checks condition first, then executes. do...while executes first, then checks condition.

üì¶Array Iteration - for...of

The for...of loop is used to iterate over iterable objects like arrays, strings, maps, sets, etc. It gives you direct access to the values.

For...of loop arrays, strings jaise iterable objects ko iterate karne ke liye use hota hai. Ye directly values deta hai.

Analogy: Like checking items in your shopping bag one by one - you take out each item (value) directly without counting positions!

Syntax:

for (let value of iterable) {
    // code to be executed
}

Example: Iterate through array

const fruits = ['Apple', 'Banana', 'Orange', 'Mango'];
for (let fruit of fruits) {
    console.log("I love " + fruit);
}

Example: Iterate through string

const name = "JavaScript";
for (let char of name) {
    console.log(char);
}
Remember: for...of gives you the values directly, not the index positions.

üóÇÔ∏èObject Iteration - for...in

The for...in loop is used to iterate over the properties (keys) of an object. It's perfect for objects where you need to access both keys and values.

For...in loop object ke properties (keys) ko iterate karne ke liye use hota hai. Ye objects ke liye perfect hai jahan keys aur values dono chahiye.

Analogy: Like checking a dictionary - you look at each word (key) and then read its meaning (value)!

Syntax:

for (let key in object) {
    // code to be executed
    // access value using object[key]
}

Example: Iterate through object

const student = {
    name: "Rahul",
    age: 20,
    course: "JavaScript",
    grade: "A"
};

for (let key in student) {
    console.log(key + ": " + student[key]);
}

Example: Array with for...in (not recommended)

const colors = ['Red', 'Green', 'Blue'];
console.log("Using for...in (gives index):");
for (let index in colors) {
    console.log(index + ": " + colors[index]);
}
Important: Use for...in for objects (gives keys), and for...of for arrays (gives values).

üõëLoop Control Statements

Loop control statements allow you to change the normal flow of loop execution. They help you exit loops early or skip certain iterations.

Loop control statements se aap loop ke normal flow ko change kar sakte ho. Ye loop ko jaldi exit karne ya kuch iterations skip karne mein help karte hain.

üö™1. Break Statement

The break statement immediately exits the loop completely, regardless of the loop condition.

Break statement loop ko turant exit kar deta hai, chahe loop condition kuch bhi ho.

Analogy: Like searching for your keys in a drawer - once you find them (condition met), you stop searching (break) immediately!

Example: Find first even number greater than 10

for (let i = 1; i <= 20; i++) {
    if (i > 10 && i % 2 === 0) {
        console.log("Found: " + i);
        break; // Exit loop immediately
    }
    console.log("Checking: " + i);
}

Example: Break in nested loop

for (let i = 1; i <= 3; i++) {
    console.log("Outer loop: " + i);
    for (let j = 1; j <= 3; j++) {
        if (j === 2) {
            console.log("  Breaking inner loop at j=" + j);
            break;
        }
        console.log("  Inner loop: " + j);
    }
}
Key Point: break only exits the innermost loop it's in, not all nested loops.

‚è≠Ô∏è2. Continue Statement

The continue statement skips the current iteration and moves to the next one. The loop doesn't exit, it just skips some code.

Continue statement current iteration ko skip kar ke next iteration pe chala jata hai. Loop exit nahi hota, bas kuch code skip ho jata hai.

Analogy: Like skipping songs on a playlist - you don't stop listening (exit loop), you just skip to the next song (continue)!

Example: Print only odd numbers

for (let i = 1; i <= 10; i++) {
    if (i % 2 === 0) {
        continue; // Skip even numbers
    }
    console.log("Odd number: " + i);
}

Example: Skip multiples of 3

for (let i = 1; i <= 15; i++) {
    if (i % 3 === 0) {
        console.log("Skipping " + i + " (multiple of 3)");
        continue;
    }
    console.log("Processing: " + i);
}

üè∑Ô∏è3. Labeled Loops

Labeled loops allow you to break or continue outer loops from inside nested loops by giving loops a name (label).

Labeled loops se aap nested loops ke andar se outer loops ko break ya continue kar sakte ho, loops ko ek naam (label) dekar.

Analogy: Like having multiple floors in a building - you can use an elevator (label) to jump directly to the ground floor (outer loop) instead of taking stairs one by one!

Example: Break outer loop from inner loop

outerLoop: for (let i = 1; i <= 3; i++) {
    console.log("Outer: " + i);
    for (let j = 1; j <= 3; j++) {
        if (i === 2 && j === 2) {
            console.log("  Breaking outer loop!");
            break outerLoop; // Breaks the outer loop
        }
        console.log("  Inner: " + j);
    }
}
console.log("Loop ended");

Example: Continue outer loop

outerLoop: for (let i = 1; i <= 3; i++) {
    console.log("Outer: " + i);
    for (let j = 1; j <= 3; j++) {
        if (j === 2) {
            console.log("  Continuing outer loop");
            continue outerLoop;
        }
        console.log("  Inner: " + j);
    }
    console.log("After inner loop");
}
Syntax: labelName: for(...) { ... break labelName; }

  • Returns the new array

üìùKey Definitions

Loop: A control structure that repeats a block of code until a specified condition is met.
Iteration: One complete execution of the loop body. Each repetition is called an iteration.
Iterator: An object that enables traversal through a collection, one element at a time.
Iterable: An object that can be iterated over (arrays, strings, maps, sets, etc.).
Loop Variable: A variable used to control the loop's execution (e.g., 'i' in for loops).
Infinite Loop: A loop that never terminates because its condition never becomes false.
Nested Loop: A loop inside another loop. The inner loop completes all iterations for each iteration of the outer loop.
Loop Control Statement: Statements (break, continue) that alter the normal flow of loop execution.
