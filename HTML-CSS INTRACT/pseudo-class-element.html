:hover

Applied when an element receives focus

.element:focus { border-color: #007bff; box-shadow: 0 0 0 3px rgba(0, 123, 255, 0.25); }

üìù Text Pseudo-elements

::first-letter

Styles the first letter of an element

This paragraph has a styled first letter. Lorem ipsum dolor sit amet, consectetur adipiscing elit. The first letter is larger and styled differently from the rest of the text.

.element::first-letter { font-size: 3em; font-weight: bold; color: #007bff; float: left; }

::first-line

Styles the first line of an element

This is the first line which will be styled differently. This is the second line and subsequent lines that will have normal styling. The first line pseudo-element only affects the first line of text.

.element::first-line { font-weight: bold; color: #007bff; font-size: 1.2em; }

::selection

Styles the selected text

Select this text to see the custom selection styling! The background and text color will change when you highlight this text.

.element::selection { background: #007bff; color: white; }

üìã Summary

Key Differences

Pseudo-classes (:)

  • Select elements in specific states
  • Use single colon (:)
  • Examples: :hover, :focus, :nth-child()
  • Style existing elements

Pseudo-elements (::)

  • Create virtual elements
  • Use double colon (::)
  • Examples: ::before, ::after, ::first-letter
  • Insert or style parts of content
