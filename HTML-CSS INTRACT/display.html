üìù Quick Summary

block = Full width, new line (like div, p, h1)

inline = Side by side, no width/height (like span, a, em)

inline-block = Side by side + width/height (like button, img)

flex = Modern flexible layout (great for navigation, cards)

grid = Grid system (perfect for complex layouts)

none = Completely hidden
